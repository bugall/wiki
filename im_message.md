### 1.部落解散

```json
  {
    type:'tribe_has_been_disbanded',
    tribeId:'number',
    tribeName:'string'
  }
```
> 不生成动态,发送消息
 
### 2.申请加入部落

  {
    type:'apply_join_tribe',

    senderName:'string',
    tribeId:'number',
    tribeName:'string',
    logicYes:'Url',
    logicNo:'Url'
